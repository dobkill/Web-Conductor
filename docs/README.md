## 当前结论

该项目整体可实现，但必须先明确以下边界：

- 插件采用 Manifest V3。
- 服务端下发的能力应以固定指令集为主，不应依赖任意远程 JS 字符串执行。
- DOM 上传需要明确普通页面结构与高敏感上下文字段的处理方式。
- 所有跨页面与跨 Tab 的操作需要经过统一调度，避免内容脚本直接承担全局状态管理。
- 用户主配置入口采用 Chrome Extension Popup 页面。
